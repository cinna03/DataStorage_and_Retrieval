# Unity ScriptableObject & PlayerPrefs Project

## Overview
This project demonstrates the use of Scriptable Objects for reusable data structures and PlayerPrefs for persistent data storage in Unity.

## Features
- Scriptable Object storing player name, level, and health.
- Control script displaying Scriptable Object data in UI.
- PlayerPrefs storing string, int, and float.
- Live updating score system using Space key.
- Bonus: Item display system with icons and titles.

## How to Test
1. Open Scene
2. Press Play
3. Press Space to update score
4. Observe UI changes
